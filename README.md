# Network Packet Sniffer with Alert System

## Overview

The Network Packet Sniffer with Alert System is a Python-based cybersecurity project that captures and analyzes network packets in real time. It monitors network traffic, logs packet information, detects suspicious activities, and generates alerts when abnormal behavior is identified.

## Features

* Real-time packet capturing
* IP Address and Port Monitoring
* Protocol Detection (TCP, UDP, ICMP)
* Traffic Logging
* Suspicious Activity Detection
* Alert Generation
* SQLite Database Support
* Traffic Summary Report

## Technologies Used

* Python 3
* Scapy
* SQLite
* Matplotlib
* Linux (Kali Linux)

## Installation

```bash
git clone https://github.com/YOUR_USERNAME/Network-Packet-Sniffer.git
cd Network-Packet-Sniffer
pip install scapy matplotlib
python3 packet_sniffer.py
```
